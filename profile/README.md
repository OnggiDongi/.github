## 🧭 서비스 소개
**따봉(DDABONG)** 은 시니어가 자신의 시간과 재능을 나누는 **세대 교류 봉사 플랫폼**입니다.  

- 🏢 **기관**은 필요한 봉사자를 쉽고 빠르게 모집할 수 있습니다.  
- 👵 **시니어**는 봉사로 의미 있는 시간을 쌓고, 사회와 연결됩니다.  

당신의 **따봉**이 모여, 세상에 **따뜻한 온기**를 더합니다. 🌱
